CSS Link (Anchor Tag)

Properties can add to CSS Link Colors according to link states
The 4 Link States are:

  • a:active - when a user has already clicked the link
  • a:hover - when a mouse pointer hovers over it
  • a:visited - visited link
  • a:link - unvisited

CSS Border

The border property specifies the style, the width, and color of an element's border.
The "border" is the short-hand syntax of the following CSS Properties:

  • border-width
  • border-style
  • border-color

CSS Margin

The margin property adds space outside the border of elements.
Margin has 4 parts namely:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

CSS Padding

CSS Padding adds space between the border and the content of an element
Specifically, it is the area between the border and the content

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
