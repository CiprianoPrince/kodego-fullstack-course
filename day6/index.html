CSS Colors

There are four value types for CSS colors

  1. Color Keywords: setting colors for html elements
  2. RBA: a css function that has parameter values which are red , green, blue rgb(red, blue, green)
  3. Hex values: values that starts with "#" hash, followed by 'rrggbb' value
  4. HSL: stands for HUE = 0 - 360, SATURATION -intensity and LIGHTNESS 0 - 100

These are the following CSS properties
  1. color : sets the color of the text
  2. text-align: is for the alignment of html text
  3. text-transform: transform the text in uppercase,lowercase,or capitalize
  4. text-decoration: sets decorative lines to emphasize points in the text
  5. text-indent: is use for indentation of the first line of each paragraph
  6. letter-spacing: spacing between chars
  7. word-spacing: spacing between words
  8. line-height: spacing vertically between linesof text
  9. direction: defines the direction of a text. it can be from left to right or vice versa
  10. text-shadow: applies a shadow to the text
  11. text-overflow: defines how overflow text of the content
  12. overflow-wrap: breaks long word and wrap them onto the next line. Used to prevent the text from overflowing from the line box
  13. word-break: sets the line breaking rules when text overflow in its content box
